# 朝暮日记

一个 Windows 端和 Android 端分开的日记软件前端原型。

## 在线页面

部署到 GitHub Pages 后，入口页会打开：

- `index.html`：客户端选择入口
- `windows/index.html`：Windows 桌面端
- `android/index.html`：Android 移动端

## 当前功能

- Windows 端：写日记、搜索、新建、自动保存、日历回顾、情绪地图、私密匣、后台同步状态
- Android 端：写日记、时间线、日历、个人页、图片/语音/私密按钮反馈

## 部署方式

这是纯前端项目，不需要服务器。上传到 GitHub 仓库后，在 GitHub Pages 里选择从 `main` 分支根目录发布即可。
